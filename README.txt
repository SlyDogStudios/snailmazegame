This is a recoding of the Snail Maze Game that was built into the Sega Master System. It can be built using ca65, ld65, and the Windows COPY command using the batch file (snail.bat).

snail.asm is the main hub file where everything is included.

The source code is not fully commented.

The sound engine was created by Drag: http://drag.wootest.net/

For more info on this game, visit http://slydogstudios.org
